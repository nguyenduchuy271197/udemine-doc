# Use Cases cho Giai đoạn 1 MVP: Nền tảng Micro-learning

## 1. Quản lý Tài khoản Người dùng

### 1.1 Đăng ký và Xác thực

- UC1.1.1: Người dùng đăng ký tài khoản mới với email
- UC1.1.2: Người dùng xác thực email
- UC1.1.3: Người dùng đăng nhập vào hệ thống
- UC1.1.4: Người dùng khôi phục mật khẩu

### 1.2 Quản lý Hồ sơ

- UC1.2.1: Người dùng cập nhật thông tin cá nhân
- UC1.2.2: Người dùng thay đổi mật khẩu
- UC1.2.3: Người dùng xem lịch sử học tập
- UC1.2.4: Người dùng quản lý thông báo

## 2. Quản lý Nội dung Học tập

### 2.1 Khóa học

- UC2.1.1: Người dùng xem danh sách khóa học
- UC2.1.2: Người dùng tìm kiếm khóa học
- UC2.1.3: Người dùng đăng ký khóa học
- UC2.1.4: Người dùng xem chi tiết khóa học
- UC2.1.5: Người dùng theo dõi tiến độ khóa học

### 2.2 Bài học

- UC2.2.1: Người dùng xem bài học
- UC2.2.2: Người dùng hoàn thành bài học
- UC2.2.3: Người dùng làm bài kiểm tra
- UC2.2.4: Người dùng xem kết quả học tập

## 3. Quản lý Nội dung (Admin)

### 3.1 Khóa học

- UC3.1.1: Admin tạo khóa học mới
- UC3.1.2: Admin chỉnh sửa khóa học
- UC3.1.3: Admin xóa/ẩn khóa học
- UC3.1.4: Admin quản lý danh mục khóa học

### 3.2 Bài học

- UC3.2.1: Admin tạo bài học mới
- UC3.2.2: Admin chỉnh sửa bài học
- UC3.2.3: Admin xóa/ẩn bài học
- UC3.2.4: Admin tạo bài kiểm tra
- UC3.2.5: Admin quản lý template bài học

## 4. Tương tác và Phản hồi

### 4.1 Đánh giá

- UC4.1.1: Người dùng đánh giá khóa học
- UC4.1.2: Người dùng đánh giá bài học
- UC4.1.3: Người dùng báo cáo lỗi nội dung

### 4.2 Analytics

- UC4.2.1: Admin xem thống kê người dùng
- UC4.2.2: Admin xem báo cáo hiệu suất khóa học
- UC4.2.3: Admin xem báo cáo lỗi
- UC4.2.4: Admin xuất báo cáo phân tích
